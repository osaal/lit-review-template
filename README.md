# Template for Literature Reviews

This is a template repository for conducting literature reviews using a processing pipeline with Zotero and Obsidian.

## Contents

-   `README.md` (this document): An overview of the repository and meta-information
-   `INSTALLATION.md`: The necessary software and tools for the pipeline

## Create a repository from this template

[See these instructions](https://docs.github.com/en/repositories/creating-and-managing-repositories/creating-a-repository-from-a-template):

1.  Press `Use this template` in the upper right corner by the file list.
2.  Follow the instructions to create a repository on your own profile.

You can also use the GitHub CLI, for example:

```bash
gh repo create --template osaal/lit-review-template
```
