# Installation

This pipeline presumes the following installed software:

-   Zotero 7.0.27
-   Obsidian 1.12.4

## Zotero

**Version:** 7.0.27

Install Zotero for your system [according to the instructions](https://www.zotero.org/download/)

> [!NOTE]
> *Check version before downloading*
>
> Zotero gives you the newest version by default, currently version 8.x. This process has not been validated for version 8, because it is not available for my system at the moment (NixOS following nixpkgs:25.11). The instructions will be updated to version 8 in the summer of 2026.
